# RoomBooking Backend

Backend aplikasi Sistem Peminjaman Ruangan Kampus.  
Dibangun dengan **ASP.NET Core**.

---
Cara Menjalankan

1. Clone repo backend:
   ```bash
   git clone https://github.com/username/RoomBookingBackend.git
   cd RoomBookingBackend
2. Jalankan backend:
   dotnet run
3. Buka Swagger:
   http://localhost:5110/swagger
